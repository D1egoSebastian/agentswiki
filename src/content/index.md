# Agent Atlas

Bienvenido a la Guia Operativa E2E para construir sistemas con agentes de inteligencia artificial.

## Que vas a encontrar

- **Agentes de IA** — Que son, como funcionan, su arquitectura y componentes clave.
- **MCP** — El protocolo que estandariza la conexion entre agentes y herramientas externas.
- **SDD** — Metodologia System-Driven Development: el ciclo completo para desarrollar con IA.
- **Despliegue** — Guia paso a paso para llevar tu proyecto a produccion.
- **Glosario** — Terminos clave explicados para principiantes.

## Como navegar

Usa el menu lateral para explorar las secciones. La homepage tiene dos puertas de entrada: una para el contenido de conocimiento y otra para la guia de despliegue.
