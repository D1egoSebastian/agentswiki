# Tech Stack & Conventions

## Core Stack
- **Framework:** Vite + Tailwind CSS (para máxima ligereza y velocidad).
- **Content:** Markdown-based (las páginas de la wiki serán archivos .md para fácil edición).
- **Icons:** Lucide-react (minimalistas y limpios).

## Conventions
- **Arquitectura:** Componentes atómicos en `/src/components`.
- **Estilo:** Seguir el skill `.agents/skills/frontend-design` priorizando tipografía sobre decoraciones [4].
- **Documentación:** Usar Context7 para cualquier duda sobre la API de Tailwind o Vite [5, 6].
