# Mission — Agent Atlas

## Proposito

Agent Atlas es la **Guia Operativa E2E** para aplicar inteligencia artificial en cualquier proyecto de software **con disciplina, evitando el vibe coding**.

El objetivo no es enseñar a "programar con IA", sino enseñar a **construir sistemas con IA de forma sistematica, repetible y verificable**. El agente es el ejecutor; el humano es el revisor.

## Principio rector

**Spec-anchored, never vibe.** Toda implementacion nace de una especificacion (spec), se verifica con gates automaticos (lint, typecheck, tests) y se documenta antes de considerarse terminada. No se acepta codigo que "se ve bien" sin verificacion.

## Estructura

Dos apartados principales:

1. **Wiki de conocimiento** — Agentes de IA, MCP, metodologia SDD (con flujo visual), buenas practicas anti-vibe-coding y glosario interactivo.
2. **Guia de despliegue** — Paso a paso desde carpeta vacia hasta produccion: esquema de proyecto, arnes del agente (AGENTS.md + spec/), build, CI/CD y deploy.

## Audiencia

Desarrolladores principiantes e intermedios que quieren usar IA sin perder el control del producto. Agent Atlas les da el arnes, la metodologia y la checklist para que cada linea generada por IA sea intencional, verificada y reversible.
