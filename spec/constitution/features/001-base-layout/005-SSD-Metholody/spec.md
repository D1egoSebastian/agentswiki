### Fase 5: Sección de "Metodología SDD" (Flujo 0 a 100)
- **Descripción:** Guía paso a paso sobre cómo desarrollar sistemas reales con IA.
- **El Ciclo SDD:**
    1. **Constitution:** Reglas generales del proyecto [12].
    2. **Specify:** Definición de requerimientos y criterios de aceptación [12].
    3. **Plan:** Diseño de la estrategia técnica [12].
    4. **Tasks:** División en pasos pequeños y verificables [1, 12].
    5. **Implement:** Ejecución en modo Build o Plan [12, 13].
    6. **Verify:** Bucle de "escribir -> testear -> corregir" [12, 14].
